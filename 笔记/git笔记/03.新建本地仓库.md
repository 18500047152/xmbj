# 2:输入命令git init 初始化仓库

# 3:git status 查看仓库文件状态,结果是红色,[是还没有提交到本地仓库]

# 5: git commit -m "第一次提交的" [-m后面的''号内,是注释内容,比如:修改了内容]

# 6:git log 提交记录
